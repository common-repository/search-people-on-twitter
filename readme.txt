== Description ==

This is a very nice plug in. This allows your site visitors to search Twitter profiles and create their Wikipedia profile for twitter.

Demo Link: http://tinyurl.com/yj3tsgc
This can be customised to suit your needs. For customisation please contact me at gaikwad.abhijit@gmail.com or visit
http://www.advertisablearena.com/blog1/twitter-profile-search/ and please enter your customisation suggestions in the comments.


Known Issues: As you know twitter API allows only 100 API calls per hour. So if we make more than 100 calls per hour the application will not work for that hour. If you get this error please mail me at gaikwad.abhijit@gmail.com or if you want I can set it up to use your twitter account. For source code please contact me.
== Installation ==

This section describes how to install the plugin and get it working.

 

1. Upload 'search_twitter_users' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Widgets and add this plug in in the sidebar and you are all set.

Or to include the widget in the side bar use follwoing code

<iframe src =”http://advertisablearena.com/twitter/search_twitter_users/search_twitter_user.html” width=”200? height=”320?>
<p>Your browser does not support iframes.</p>
</iframe>

Please include a link to http://www.advertisablearena.com
